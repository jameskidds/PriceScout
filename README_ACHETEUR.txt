  PREMIERE UTILISATION — ACTIVATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Decompressez le dossier ZIP ou vous voulez (Bureau, Documents...)

  2. Double-cliquez sur  PriceScout.exe

  3. Une fenetre d'activation s'ouvre — entrez la cle que vous
     avez recue avec votre achat
     (format : PSCT-XXXXXXXX-XXXXXXXX)

  4. Cliquez "Activer" -> "Activation reussie !"

  5. Votre navigateur s'ouvre automatiquement avec l'application.

  L'activation est definitive sur votre PC.
  La prochaine fois, double-cliquez simplement sur l'exe,
  l'application s'ouvre directement sans redemander la cle.

  ATTENTION : La cle est liee a votre ordinateur.
  Elle ne fonctionnera pas sur un autre PC.

  OPTION — ALERTES SUR TELEPHONE VIA TELEGRAM
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Recevez vos alertes directement sur votre telephone.
  Gratuit, aucune inscription payante.

  ETAPE 1 : Creer votre bot Telegram
  ────────────────────────────────────
  1. Telechargez Telegram sur votre telephone (gratuit)
     et creez un compte si vous n'en avez pas.

  2. Dans la barre de recherche Telegram, cherchez :
        @BotFather
     (il a une coche bleue officielle)

  3. Tapez :  /newbot

  4. Il vous demande un NOM pour votre bot
     -> Exemple : Mes Alertes Prix

  5. Il vous demande un IDENTIFIANT (doit finir par "bot")
     -> Exemple : mesalertesprix_bot

  6. BotFather vous envoie votre TOKEN. Il ressemble a :
        123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
     COPIEZ CE TOKEN, vous en aurez besoin a l'etape 3.

  ETAPE 2 : Obtenir votre Chat ID
  ────────────────────────────────
  1. Dans Telegram, cherchez :  @userinfobot

  2. Tapez :  /start

  3. Il vous repond avec votre Chat ID (un nombre, ex: 123456789)
     COPIEZ CE NOMBRE.

  ETAPE 3 : Connecter Telegram a Price Scout
  ────────────────────────────────────────────
  1. Ouvrez Price Scout dans votre navigateur

  2. Cliquez sur l'onglet  PARAMETRES (roue crantee)

  3. Dans "Token du bot"  : collez votre token
     Dans "Chat ID"       : collez votre Chat ID

  4. Cliquez "Tester"
     -> Vous recevez un message Telegram :
        "Price Scout — connexion Telegram active !"

  5. Cliquez "Sauvegarder"

  Termine ! Des qu'une alerte se declenche, vous recevez
  un message Telegram avec le lien et le prix de l'annonce.

  Q : Windows bloque le fichier au premier lancement ?
  R : Cliquez "Informations complementaires" puis "Executer quand
      meme". C'est normal pour un .exe non signe, sans danger.

  Q : Mon antivirus bloque le fichier ?
  R : Ajoutez PriceScout.exe en exception dans votre antivirus.

  Q : Je change de PC, ma cle fonctionne encore ?
  R : Non, contactez le vendeur pour un transfert.

  Q : Le test Telegram echoue ?
  R : Verifiez que le token est copie en entier et que vous avez
      appuye sur Demarrer dans votre bot Telegram avant de tester.
