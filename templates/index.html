📖 Comment créer votre bot Telegram ?

1
Ouvrez Telegram et cherchez @BotFather
2
Tapez /newbot → choisissez un nom (ex: Mes Alertes Prix) → choisissez un identifiant (ex: mesalertesprix_bot)
3
BotFather vous envoie un token — copiez-le, c'est votre Token du bot (ressemble à 123456789:AAHxxxx...)
4
Cherchez votre bot dans Telegram et cliquez Démarrer
5
Cherchez @userinfobot → tapez /start → il vous envoie votre Chat ID (un nombre, ex: 123456789)
6
Collez le token et le Chat ID ci-dessous → cliquez Tester puis Sauvegarder
